Tech 3351 v2.2 EBU CCDM 35 Figure 8: Consumption Domain 2.2.6.1 ConsumptionEvent Definition: Represents the event of a user consuming a published content. A ConsumptionEvent follows publication but is no longer related to the PublicationEvent. The link to the PublicationEvent is represented via the Essence it consumes. For linear services the ConsumptionEvent and PublicationEvent happen at the same time (well, almost, when respecting signal transport and transformation time). For non-linear services, the Consumer decides about the time of the ConsumptionEvent.
